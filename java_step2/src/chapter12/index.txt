추상 클래스(abstract class)

클래스가 설계도라면 추상클래스는 미완성 설계도.
엄밀하게는 : 추상-메서드를 포함하는 클래스를 추상 클래스라고 한다.

특) 인스턴스 생성이 불가능.
상속받은 서브클래스에서 추상메서드를 구현하고 해당 서브클래스의 객체를 통해 구현된다.

int add(int n1, intn2){}; << 구현이 됐다.(물론 리턴이 없다고 컴파일에러 뜨고 그러겠죠) 이건 추상-메서드가 아니다.
abstract int add(int n1, int n2); << 구현이 되지 않았다. 이걸 상속받은 서브클래스에서, 저 이름과 인자의 수, 종류 그대로 로직을 구현한다.
