Wrapper 클래스
	Java에서 사용하는 기본 타입의 변수들을 "참조형" 변수로 바꾸기 위한 클래스이다.

int -> Integer
byte -> Byte
char -> Character
long -> Long
short -> Short
double -> Double
float -> Float

기본 타입명을 첫 글자만 대문자로 바꿔서 정의해놓는 규칙이 있다.

왜 필요한가요?
	-> 기본 타입은 "값"만 저장하며 stack에 생성된다.
	-> 참조 타입(=대상의 주소 값. Heap 메모리에 생성된다)을 요구 할 때 기본 타입을 넣을 수 없다는 문제가 있다.

박싱 : 기본타입 -> Wrapper		int a = 3; Integer wrappedA = Integer(a);
언박싱 : Wrapper -> 기본타입
