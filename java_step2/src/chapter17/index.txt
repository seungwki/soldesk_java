Full name : "Collection Framework"
	기본적으로 프레임워크이다

collection(interface)
	List(interface) : 순서 보장 o, 중복 o
		Array List
		Linked List
		Vector(Deprecated)
	Queue(interface) : First-in, First-Out(FIFO==선입선출).
		Deque(interface)
		ArrayDeque
	Set(interface) : 순서 보장 x, 중복 x
		HashSet
			Linked Hashset
		TreeSet
	
Map(interface) : Key-Value 구조. 순서x, 키-중복x, 값-중복o
	HashMap : 순서 x
		LinkedHashMap : 입력순서 o
	TreeMap : 정렬 상태를 유지.
	Properties(Deprecated)
