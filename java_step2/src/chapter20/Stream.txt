스트림
	배열, 컬렉션 등을 더 쉽고 빠르게 처리하고 변환할 때 사용한다
	일관성있게 처리 할 수 있는 메서드를 제공한다.
	일회용이다.
	기존 자료에 영향을 미치지 않는다.
	중간 연산과 최종 연산이 있다
		최종 연산이 호출 되지 않으면 정렬이나 검색한 결과를 가져올 수 없음
		중간연산 : filter() 조건을 넣고 참인 것들을 추출, map() 스트림의 각 요소를 새 요소로 변환
		최종연산 : 자료를 소모하며 연산을 수행
			forEach()
			count()
			sum()
			reduce()
