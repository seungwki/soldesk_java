▶멀티프로세스
	== 운영체제. 여러 개의 프로세스를 가질 수 있다.
	▶프로세스
	독립적 메모리 공간을 가지며, 서로 간섭하지 않는다.
	운영체제가 별도로 작업 단위를 관리한다.
	프로세스 간에는 직접적 메모리 접근이 불가능하다.
		▶스레드
		프로세스 내에서 실행되는 작업의 단위.
		프로세스는 최소 하나 이상의 스레드를 갖는다.(싱글 스레드 또는 멀티 스레드)
		스레드는 프로세스보다 단순하므로 생성 및 관리가 단순하고 가볍다.
		▶프로세스의 메모리 구성
		공유 스택
			코드
			데이터
			스택
			힙
		개별 스택
			스레드n(스택영역)(n>=1)
